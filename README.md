# 주식 매수/매도 계산 도구

웹 기반 주식 매수/매도 계산 도구입니다.

## 기능
- 실시간 주가 조회 (야후 파이낸스)
- 매수 가능 주수 계산
- 매도 손익/수익률 계산
- 포트폴리오 현황 관리
- PC/모바일 PWA 지원

## 기술 스택
- React 18 + TypeScript
- Tailwind CSS
- Firebase Firestore
- Zustand (상태 관리)

## 설치
\\\ash
npm install
npm run dev
\\\

## 환경 설정
\.env.local\ 파일 생성:
\\\
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_PROJECT_ID=...
\\\

## 배포
\\\ash
npm run build
